## What is Defensive Security?

Defensive security focuses on:
- detecting cyber attacks,
- preventing unauthorized access,
- monitoring systems,
- and responding to security incidents.

The goal is to protect systems and maintain security.

---

## Security Operations Center (SOC)

A SOC is a team responsible for continuously monitoring and improving an organization’s security posture.

### SOC Tasks:
- Monitor network traffic
- Analyze logs
- Detect threats
- Investigate incidents
- Respond to attacks

---

## Important Security Concepts

### Logs
Logs are records of events generated by systems and applications.

Examples:
- Login attempts
- Failed authentications
- System errors
- Network activity

---

### Alerts
Alerts are notifications generated when suspicious activity is detected.

Examples:
- Multiple failed login attempts
- Malware detection
- Unusual network behavior

---

### Incident Response
Incident response is the process of handling security incidents effectively.

Basic Steps:
1. Detect
2. Analyze
3. Contain
4. Eradicate
5. Recover

## Key Takeaways

- Defensive security is essential for protecting organizations.
